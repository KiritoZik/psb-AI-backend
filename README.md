# PSB AI Backend

Backend для обработки писем с использованием ML классификации.

## Описание

Система автоматической обработки входящих писем с генерацией ответов на основе ML-классификации. Позволяет:
- Автоматически классифицировать письма по типу и срочности
- Генерировать ответы с учетом контекста письма
- Управлять письмами через админ-панель (просмотр, редактирование, одобрение, отправка)
- Отслеживать историю обработки писем

## Быстрый старт

1. **Установка зависимостей:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Создание .env файла:**
   ```bash
   ADMIN_PASSWORD=ваш-пароль
   ```

3. **Запуск сервера:**
   ```bash
   python main.py
   ```
   
   Сервер: http://localhost:8000

## API эндпоинты

**Публичные:**
- `POST /api/v1/process-letter` - Обработка письма
- `POST /api/v1/generate` - Генерация ответа

**Защищенные (требуют авторизации):**
- `POST /api/v1/admin/auth/login` - Авторизация
- `GET /api/v1/admin/letters` - Список писем
- `GET /api/v1/admin/letters/{id}` - Детали письма
- `PUT /api/v1/admin/letters/{id}/edit` - Редактирование
- `POST /api/v1/admin/letters/{id}/approve` - Одобрение
- `POST /api/v1/admin/letters/{id}/send` - Отправка
- `GET /api/v1/admin/history` - История
